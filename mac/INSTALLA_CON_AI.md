# Installa con Claude Code o Codex

Questa e' la strada piu' semplice.

## Cosa devi fare

1. Apri Claude Code o Codex sul tuo Mac.
2. Copia il prompt qui sotto.
3. Incollalo nell'agente.
4. Segui le istruzioni a schermo.

## Prompt da copiare

```text
Voglio installare Voce Dettatura per Mac da questo repository, cartella mac/:
https://github.com/salChiarenza/voce

Guidami passo passo.

Prima controlla se il mio Mac e' compatibile.
Poi scarica il progetto, installa le dipendenze, crea i launcher sulla Scrivania e fammi fare una prova reale di dettatura.

Quando macOS chiede permessi, spiegami esattamente cosa devo cliccare.
Non saltare i controlli: alla fine voglio sapere se la dettatura funziona davvero.
```

## Cosa fara' l'agente

- Controlla il Mac.
- Scarica il repository.
- Esegue l'installer.
- Crea i launcher sulla Scrivania.
- Ti guida nei permessi di macOS.
- Fa una prova finale di dettatura.

## Se si blocca

Scrivi nella community AI con Sal:

- modello del Mac;
- versione macOS;
- quale passaggio si blocca;
- cosa vedi a schermo.
